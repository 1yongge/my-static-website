# 自由迪迪 - 现代博客网站

一个现代化、响应式的博客网站，模仿 freedidi.com 的设计风格，具有完整的博客功能和管理后台。

## 网站特性

### 前端功能
- **现代化设计**：采用最新的UI/UX设计理念
- **响应式布局**：完美适配桌面、平板和移动设备
- **丰富的交互**：平滑动画和过渡效果
- **博客系统**：完整的博客文章展示功能
- **分类和标签**：灵活的内容组织方式
- **搜索功能**：便捷的内容查找
- **社交集成**：多平台社交媒体链接

### 管理后台
- **仪表板**：实时数据统计和概览
- **文章管理**：创建、编辑、删除文章
- **分类管理**：灵活的内容分类系统
- **用户管理**：用户权限和角色管理
- **评论管理**：评论审核和回复
- **媒体库**：图片和文件管理
- **网站设置**：全局配置选项

## 技术栈

- **前端**: HTML5, CSS3, JavaScript ES6+
- **样式**: 现代CSS (Flexbox, Grid, CSS Variables)
- **图标**: Font Awesome 6
- **架构**: 移动优先的响应式设计
- **性能**: 优化的加载速度和用户体验

## 项目结构

```
my-static-website/
├── index.html              # 首页
├── blog.html               # 博客列表页
├── post.html               # 文章详情页
├── admin.html              # 管理后台
├── categories.html         # 分类页面
├── tags.html               # 标签页面
├── about.html              # 关于页面
├── assets/
│   ├── css/
│   │   ├── main.css        # 主样式
│   │   ├── blog.css        # 博客页面样式
│   │   └── admin.css       # 管理后台样式
│   └── js/
│       ├── main.js         # 主要功能脚本
│       └── admin.js        # 管理后台脚本
├── robots.txt              # 搜索引擎配置
└── sitemap.xml             # 网站地图
```

## 部署到GitHub Pages

1. 将此仓库推送到GitHub
2. 在仓库设置中启用GitHub Pages
3. 选择main分支作为源
4. 网站将在 `https://<username>.github.io/<repository>` 上可用

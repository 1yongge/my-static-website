# 自由迪迪 - 个人博客网站

这是一个静态博客网站，包含前端和管理后台功能。

## 网站特性

### 前端功能
- 响应式设计，支持各种设备
- 博客文章展示
- 分类浏览
- 文章详情页
- 关于页面
- 搜索功能
- 社交分享

### 管理后台
- 仪表板统计
- 文章管理（创建、编辑、删除）
- 分类管理
- 用户友好的界面

## 技术栈

- HTML5
- CSS3 (包含响应式设计)
- JavaScript (ES6+)
- 静态网站结构

## 项目结构

```
my-static-website/
├── index.html          # 首页
├── blog.html           # 博客列表页
├── post.html           # 文章详情页
├── admin.html          # 管理后台
├── about.html          # 关于页面
├── categories.html     # 分类页面
├── css/
│   ├── style.css       # 主样式
│   ├── blog.css        # 博客页面样式
│   └── admin.css       # 管理后台样式
├── js/
│   ├── main.js         # 主要功能脚本
│   └── admin.js        # 管理后台脚本
├── robots.txt          # 搜索引擎配置
└── sitemap.xml         # 网站地图
```

## 使用方法

1. 克隆此仓库
2. 在浏览器中打开 `index.html`
3. 访问 `admin.html` 进入管理后台

## 部署到GitHub Pages

1. 将此仓库推送到GitHub
2. 在仓库设置中启用GitHub Pages
3. 选择main分支作为源
4. 网站将在 `https://<username>.github.io/<repository>` 上可用

## 功能说明

### 前端功能
- 导航栏包含首页、博客、分类、关于和管理后台链接
- 首页展示特色文章和热门分类
- 博客页面展示文章列表和支持侧边栏
- 文章详情页面包含完整内容和相关文章推荐

### 管理后台功能
- 仪表板显示统计数据
- 文章创建表单
- 文章管理表格（编辑、删除）
- 实时字符计数器

## 设计理念

此网站采用了现代化的设计风格，注重用户体验和易用性。布局简洁明了，色彩搭配舒适，确保内容易于阅读。

## 自定义

您可以轻松自定义：

- 更新JavaScript以添加新功能
- 修改HTML内容以适应您的需求

# 我的静态网站

这是一个简单的静态网站示例，展示了基本的HTML、CSS和JavaScript功能。

## 功能特性

- 响应式设计
- 平滑滚动导航
- 交互式元素
- 现代化的样式

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6)

## 如何使用

1. 克隆此仓库
2. 在浏览器中打开 `index.html` 文件
3. 浏览网站的不同部分

## 项目结构

```
├── index.html
├── styles.css
├── script.js
└── README.md
```

## 部署

此项目可以很容易地部署到GitHub Pages、Netlify或其他静态网站托管服务。
